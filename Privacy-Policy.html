Privacy Policy

Invoice Generator for Freelancers

Last Updated: April 15, 2026

1. Overview

Invoice Generator for Freelancers is designed to help users create, manage, and export invoices easily. Your privacy is important, and this policy explains how your data is handled within the app.

2. Data We Collect

2.1 Personal Information

The app collects user-entered information such as:

  • Client name
  • Email address
  • Phone number
  • Physical address

2.2 Invoice Information

  • Invoice items and descriptions
  • Pricing and totals
  • Notes and payment details (if added by user)

3. How Data is Used

All collected data is used strictly for app functionality, including:

  • Creating and managing invoices
  • Generating invoice PDFs
  • Storing client records locally
  • Displaying invoice history within the app

4. Data Storage

  • All data is stored locally on your device only
  • No cloud storage or external servers are used
  • Data is saved using Android’s private storage system

5. Data Sharing

We do not share, sell, or transfer any user data to third parties.

  • No external servers receive user data
  • No analytics or marketing platforms receive personal information
  • Data remains fully on the user’s device

6. Data Collection & Transmission

  • User data is collected only for app functionality
  • Data is not transmitted to external servers
  • Data is not shared with third parties

7. Permissions

  • No location access
  • No contacts access (unless manually entered by user)
  • No camera access
  • No unnecessary background data collection

8. Third-Party Services

This app does not use:

  • Analytics tools
  • Advertising networks (unless added later)
  • External APIs
  • Tracking services

9. Data Security

Your data is stored securely using Android’s built-in app sandboxing and secure local storage. No internet transmission is used, minimizing security risks.

10. Children’s Privacy

This app is suitable for general use. We do not knowingly collect additional data from children beyond user-entered information required for app functionality.

11. User Control

  • You can edit or delete invoices anytime inside the app
  • Uninstalling the app will permanently delete all stored data

12. Changes to This Policy
